# XYZ Blogs - CMS & Blog Platform

XYZ Blogs is a complete, custom-built Content Management System (CMS) and blogging platform powered by raw PHP, MySQLi, and Bootstrap 5. It is designed to be lightweight, easy to host, and fully responsive, featuring a modern and dynamic UI.

## Features

- **User Authentication:** Secure registration and login system utilizing `password_hash()` and `password_verify()`.
- **User Profiles:** Users get personalized profiles with custom biographies and avatar uploads.
- **Newsfeed:** A global, chronologically ordered feed where all user posts are displayed.
- **Post Management:** Authenticated users can create, edit, and delete their own posts. Support for image uploads on posts is included.
- **Interactivity (Likes):** Users can seamlessly "like" or "unlike" posts in real-time without page reloads using jQuery AJAX.
- **Global Search:** Find other users or specific blog posts via a central search engine.
- **Modern UI:** Built on top of Bootstrap 5 with custom CSS to provide subtle micro-interactions, hover effects, and a premium look.

## Technologies Used

- **Backend:** PHP 8+
- **Database:** MySQL / MariaDB (MySQLi extension)
- **Frontend:** HTML5, CSS3, JavaScript (jQuery for AJAX)
- **CSS Framework:** Bootstrap 5.3

## Setup and Installation (Local Environment)

Follow these steps to run the application on your local machine using a server environment like XAMPP, WAMP, or Laragon.

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/antickbhattacharjee/XYZ_Blogs.git
   ```
2. **Move to Server Directory:** 
   Place the cloned `XYZ_Blogs` folder inside your web server's root directory (e.g., `htdocs` for XAMPP or `www` for WAMP).
3. **Start Apache and MySQL:**
   Ensure your local server's Apache and MySQL services are actively running.
4. **Database Configuration:**
   Open the `config.php` file in the root directory and ensure the credentials match your local MySQL setup (usually `root` with a blank password).
   ```php
   define('DB_HOST', 'localhost');
   define('DB_USER', 'root');
   define('DB_PASS', '');
   define('DB_NAME', 'blog_db');
   ```
5. **Run Setup Script:**
   Navigate to the automated database setup script in your browser to automatically generate the database and required tables:
   ```
   http://localhost/XYZ_Blogs/setup_db.php
   ```
6. **Start Using the App!**
   Click the link generated by the setup script, or navigate to `http://localhost/XYZ_Blogs/index.php`. Register a new account and start blogging!

## Setup for Live Web Hosting (cPanel, etc.)

1. **Upload Files:** Upload the entire contents of this repository to your web host's `public_html` directory via FTP or File Manager.
2. **Create Live Database:** Using your hosting control panel, create a new MySQL database and a new user. Assign the user all privileges to the database.
3. **Update config.php:** Open the `config.php` file on your live server and input the live database name, user, and password.
4. **Run Setup:** Visit `https://yourdomain.com/setup_db.php` to initialize the tables on your live database.
5. **Security Cleanup:** Once the setup is complete and successful, it is highly recommended to **delete** `setup_db.php` and `schema.sql` from your live server to prevent unwanted database resets.

```text
├── .htaccess             # Basic Apache security rules
├── config.php            # Database credentials configuration
├── index.php             # Main newsfeed
├── login.php             # User login
├── register.php          # User registration
├── logout.php            # Ends user session
├── profile.php           # User profile and posts viewer
├── edit_profile.php      # Form to update bio and avatar
├── create_post.php       # Form to create a blog post
├── edit_post.php         # Form to modify an existing post
├── delete_post.php       # Logic to delete a post
├── post.php              # Single post view
├── search.php            # Global search logic and view
├── like_post.php         # AJAX endpoint for liking posts
├── schema.sql            # Raw database schema
├── setup_db.php          # Automated setup script
│
├── includes/
│   ├── db.php            # Database connection initializer
│   ├── functions.php     # Helper functions (sanitization, auth checks)
│   ├── header.php        # Global HTML head and Navbar
│   └── footer.php        # Global HTML footer and JS imports
│
└── assets/
    ├── css/
    │   └── style.css     # Custom application styling
    ├── js/
    │   └── script.js     # AJAX handlers
    └── uploads/          # Directory for dynamic user-uploaded images
```
